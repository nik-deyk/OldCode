Чтобы принять участие, пройдите регистрацию и _решите один номер в контесте_.

### Проверить у себя:

Вам понадобятся `python >=3` и `java >=15`. Скомпилируйте java-класс:

```
javac A.java
```

Сгенерируйте тест:

```
python generate.py
```

Выполните программу:

```
cat test.txt | java A
```
